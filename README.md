### Instruções de inicialização

Para iniciar o projeto, siga os passos abaixo:

1. **Acesse a pasta `back-end`**:

   ```bash
   cd back-end

2. **Instale as dependências do projeto** 

    ```bash
    npm install

3. **Inicie o servidor**

    ```bash
    npm run dev

4. **Acesse o link:

    Abra o navegador e acesse o link: http://localhost:8000/
